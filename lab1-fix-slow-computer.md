# Lab 1 — Fix a Slow Computer

## Problem

Computer running very slow while using Google Chrome.

## Investigation

Opened Task Manager to analyze system performance.

## Findings

Google Chrome was using around 85% of system memory due to many open tabs.

## Steps Taken

1. Opened Task Manager (Ctrl + Shift + Esc)
2. Sorted processes by Memory usage
3. Identified Chrome as the heavy process
4. Closed unnecessary browser tabs

## Result

Memory usage dropped from 85% to 45%.
System performance improved.

## Skills Practiced

* Windows Troubleshooting
* Task Manager analysis
* Identifying resource heavy processes
